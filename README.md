# CPX Protocol — Official Website

> **"CPX Protocol is the economic security layer. SHIELD-X is the operational risk intelligence engine."**

## Overview

This repository contains the official website for **CPX Protocol** — a behavioral risk research initiative built to protect retail traders in digital asset markets.

The website is a static multi-page HTML site deployed on Vercel.

| Page | Description |
|------|-------------|
| `index.html` | Landing page — Hero, Problem, Solution, Business Model, Roadmap, Founder |
| `dashboard.html` | SHIELD-X Beta Dashboard — Mock institutional risk intelligence interface |
| `research.html` | Research framework, module architecture, whitepaper link |
| `about.html` | Founder story, vision, CPX token role |
| `contact.html` | Business inquiry form (Formspree) |
